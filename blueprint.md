# 프리미엄 로또 추천기 개발 블루프린트

## 1. 프로젝트 개요

- **프로젝트 명:** 프리미엄 로또 추천기
- **목표:** 단순한 로또 번호 생성을 넘어, 통계 데이터, 흥미로운 콘텐츠, 세련된 사용자 경험을 결합하여 건전하고 즐거운 로또 문화를 선도하는 정보 제공 웹사이트 구축.
- **핵심 기능:**
    - 통계 기반 로또 번호 생성 (웹 컴포넌트 활용)
    - 로또 관련 정보 및 스토리 제공 (블로그)
    - 깔끔하고 직관적인 UI/UX
    - 다크 모드 지원

---

## 2. 스타일 및 디자인 가이드

- **폰트:** 가독성이 높은 'Poppins'와 시스템 기본 산세리프 폰트 사용
- **색상 팔레트:**
    - **Primary:** `#fbc400` (골드 옐로우 - 행운, 긍정)
    - **Background (Dark):** `#1a1a1a` (깊이감 있는 다크 그레이)
    - **Background (Light):** `#ffffff` (깨끗한 화이트)
    - **Text (Dark):** `#e0e0e0` (부드러운 라이트 그레이)
    - **Text (Light):** `#333333` (진한 그레이)
    - **Accent Colors:** `#69c8f2`, `#ff7272`, `#b0d840` (로또볼 색상)
- **디자인 특징:**
    - **Glassmorphism:** 반투명한 배경과 그림자를 활용하여 깊이감과 세련미 강조 (카드 UI)
    - **미니멀리즘:** 불필요한 장식을 최소화하고 콘텐츠에 집중할 수 있는 레이아웃
    - **아이콘:** 의미를 명확하게 전달하는 이모지 아이콘 활용 (`✨`, `🎯`, `📊` 등)
    - **애니메이션:** 부드러운 전환(`transition`) 효과와 미묘한 호버(`:hover`) 효과로 사용자 경험 향상

---

## 3. 구현된 기능 및 페이지

- **`index.html` (홈페이지):**
    - 프로젝트의 정체성을 보여주는 Hero 섹션
    - 사이트의 목적과 콘텐츠를 소개하는 환영 메시지 섹션 **(콘텐츠 보강 완료)**
    - `lotto-generator` 웹 컴포넌트를 이용한 번호 생성 기능
    - 로또 기본 정보, 통계 분석, 건전 문화 캠페인 섹션
- **`about.html` (소개):**
    - 사이트의 미션과 비전, 운영 철학 소개
    - 운영팀의 스토리를 추가하여 신뢰도 강화 **(콘텐츠 보강 완료)**
- **`blog.html` (블로그 목록):**
    - 발행된 모든 블로그 게시물을 보여주는 카드형 목록
- **`blog-post-*.html` (블로그 게시물):**
    - 개별 블로그 콘텐츠 페이지
    - 일관된 상단 네비게이션 적용 (사이드 메뉴 제거 완료)
- **`contact.html`, `privacy.html`, `terms.html`:**
    - 사이트 신뢰도를 위한 필수 정보 페이지
- **`coin-flip.html`:**
    - 재미 요소를 위한 미니 게임
- **공통 기능:**
    - **다크 모드:** `localStorage`를 이용한 테마 저장 기능
    - **반응형 디자인:** 모바일 및 데스크톱 환경 최적화

---

## 4. 콘텐츠 전략

- **목표:** 로또와 관련된 유용하고 흥미로운 정보 제공으로 사이트의 전문성과 가치 증대
- **발행 현황:**
    - **게시물 1:** 로또 당첨의 심리학: 왜 우리는 번호를 선택할까?
    - **게시물 2:** 역대 최고 로또 당첨금: 인생을 바꾼 행운의 순간들
    - **게시물 3:** 로또 초보자를 위한 완벽 가이드: 구매부터 당첨금 수령까지
    - **게시물 4:** 만약 로또에 당첨된다면? 현명한 당첨금 사용 가이드
    - **게시물 5:** 로또 번호의 비밀: 통계로 풀어보는 당첨 확률 높이기 **(신규 발행)**
    - **게시물 6:** 로또 당첨금의 세금, 얼마나 뗄까? (2024년 기준) **(신규 발행)**

## 5. 애드센스 승인 계획 (v1 - 2024.05)

- **이슈:** '콘텐츠 없음' 사유로 1차 반려
- **해결 전략:** 사이트 전반의 텍스트 콘텐츠 양을 늘리고, 각 페이지의 목적과 정체성을 명확하게 하여 사이트의 가치를 증명한다.

- **실행 완료 내역:**
    1.  **홈페이지 콘텐츠 보강:** `index.html`에 사이트의 목적, 비전, 콘텐츠를 소개하는 설명 섹션을 추가함.
    2.  **소개 페이지 강화:** `about.html`에 운영자의 스토리와 미션을 구체적으로 서술하여 신뢰도를 높임.
    3.  **신규 블로그 2건 발행:** 통계 분석, 세금 정보 등 전문성과 유용성을 갖춘 신규 콘텐츠(`blog-post-5.html`, `blog-post-6.html`)를 추가하여 전체 콘텐츠 양을 늘림.
    4.  **블로그 목록 업데이트:** `blog.html`에 신규 게시물 2건을 추가함.

- **향후 계획:**
    - 위 변경사항을 적용하여 구글 애드센스 재심사 요청
    - 지속적인 양질의 콘텐츠 발행 (월 2-4회)
