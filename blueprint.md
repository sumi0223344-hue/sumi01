
# 프로젝트 개요

프리미엄 로또 추천기는 사용자에게 로또 번호 조합을 추천하고 관련 정보를 제공하는 웹 서비스입니다. 이 서비스는 단순한 무작위 번호 생성을 넘어, 통계적 데이터와 사용자 경험을 결합하여 로또를 더욱 즐겁고 흥미롭게 즐길 수 있도록 돕는 것을 목표로 합니다. 또한, 건전한 로또 문화를 장려하고 사용자에게 유용한 정보를 제공하여 신뢰할 수 있는 플랫폼으로 자리매김하고자 합니다.

# 디자인 및 기능 명세

## 1. 공통 스타일 및 레이아웃

- **디자인:** 전반적으로 어두운 배경과 밝은 텍스트를 기본으로 하는 다크 모드 디자인을 적용하여 눈의 피로를 줄이고 고급스러운 느낌을 줍니다. 라이트 모드 옵션도 제공하여 사용자 선택권을 보장합니다.
- **아이콘:** 각 페이지 제목과 주요 기능에 관련된 아이콘을 추가하여 페이지의 주제를 시각적으로 명확하게 전달하고 사용자 인터페이스의 직관성을 높였습니다.
- **글꼴:** 가독성이 높은 'Noto Sans KR'와 'Poppins' 글꼴을 사용하여 콘텐츠의 전달력을 높였습니다.
- **색상:** 행운과 신뢰를 상징하는 색상 조합을 사용하여 사용자에게 긍정적인 인상을 줍니다.
- **반응형 디자인:** 모바일, 태블릿, 데스크톱 등 모든 기기에서 최적화된 화면을 제공합니다.
    - **반응형 네비게이션:** 데스크톱에서는 상단 네비게이션 메뉴를, 모바일 및 태블릿에서는 숨겨진 사이드 네비게이션(햄버거 메뉴)을 제공하여 화면 공간을 효율적으로 사용하고 터치 조작성을 향상시켰습니다.
- **버튼 스타일:** 모든 페이지에서 일관된 버튼 스타일('lotto-btn')을 적용하여 시각적 통일성을 확보하고 사용자에게 일관된 경험을 제공합니다.

## 2. 페이지별 기능 및 개선 사항

- **메인 페이지 (index.html):**
    - **오늘의 행운 번호:** 사용자가 번호를 생성할 때마다 5개의 추천 조합과 함께 특별한 행운 번호 하나를 추가로 제공하여 재미와 기대감을 높입니다.
    - **개선:** 기본 CTA 버튼을 더욱 눈에 띄게 만들어 사용자 참여를 유도했습니다.
    - 로또 번호 생성기: 5개의 추천 조합을 제공하며, 버튼 클릭 시 새로운 조합으로 갱신됩니다.
    - 블로그 섹션: 최근 블로그 게시물 3개를 소개해 사용자의 흥미를 유발하고 사이트 내 체류 시간을 늘립니다.
    - 로또 정보: 로또 게임 방법, 추첨 시간, 통계 분석 등 유용한 정보를 제공합니다.
- **소개 페이지 (about.html):**
    - **개선:** 미션과 비전 섹션을 명확하게 구분하여 서비스의 핵심 가치를 효과적으로 전달합니다.
    - 서비스의 비전과 철학: 데이터 기반 접근과 인간적인 감성을 결합한 서비스의 차별점을 소개합니다.
    - 신뢰와 투명성 강조: 정보의 출처와 서비스의 한계를 명확히 밝혀 사용자의 신뢰를 얻습니다.
- **블로그 페이지 (blog.html):**
    - **개선:** 블로그 게시물 요약을 시각적으로 더 매력적으로 만들어 사용자의 흥미를 끌고 가독성을 높였습니다.
    - 블로그 게시물 목록: 작성된 모든 블로그 게시물을 리스트 형태로 보여주어 사용자가 원하는 정보를 쉽게 찾을 수 있도록 합니다.
- **블로그 게시물 페이지 (blog-post-*.html):**
    - **개선:** 각 블로그 게시물 제목에 주제에 맞는 아이콘(🌟, 🏆, 🎓)을 추가하여 시각적 매력을 더하고 콘텐츠의 성격을 쉽게 파악할 수 있도록 했습니다.
    - 깊이 있는 콘텐츠: 로또와 관련된 다양한 주제(심리학, 역사, 가이드 등)를 심도 있게 다루어 사용자에게 유용한 정보를 제공합니다.
- **문의하기 페이지 (contact.html):**
    - **개선:** '문의하기'와 '자주 묻는 질문(FAQ)' 섹션을 아이콘(✉️, ❓)과 함께 명확하게 구분하여 사용자가 원하는 정보를 쉽게 찾을 수 있도록 했습니다.
    - 문의 채널 안내: 이메일 주소, 문의 처리 시간 등 명확한 정보를 제공하여 사용자와의 소통을 원활하게 합니다.
- **동전 던지기 페이지 (coin-flip.html):**
    - **개선:** 페이지의 스타일 코드를 `style.css`로 통합하여 코드의 일관성과 유지보수성을 높였습니다. 제목과 버튼에 아이콘(🪙, 🔄)을 추가하고, 버튼 스타일을 다른 페이지와 통일했습니다.
    - 추가 기능: 사용자가 간단하게 즐길 수 있는 미니 게임을 제공하여 재미 요소를 더합니다.
- **개인정보처리방침 및 이용약관 페이지 (privacy.html, terms.html):**
    - **개선:** 제목에 아이콘(🔒, 📜)을 추가하고, 법률 조항의 가독성을 높이기 위해 텍스트 서식을 개선하여 사용자가 내용을 더 쉽게 이해할 수 있도록 했습니다.
    - 법적 고지 사항: 서비스 운영에 필요한 법적 내용을 명시하여 투명성을 확보합니다.

# 현재까지의 진행 상황

- **1차:** 다크 모드, 푸터, 기본 페이지(홈, 소개, 문의) 구조 완성
- **2차:** 애드센스 승인을 위한 콘텐츠 보강 (블로그, 개인정보처리방침, 이용약관 페이지 추가)
- **3차:** 사용자 편의성을 위한 기능 추가 (동전 던지기 페이지, 아이콘, 즐겨찾기 아이콘)
- **4차:** 모바일 반응형 디자인 적용 (전체 페이지에 햄버거 메뉴 및 사이드 네비게이션 도입)
- **5차:** 웹사이트 전체 디자인 시스템 통합 및 고도화
    - 모든 페이지에 일관된 아이콘, 버튼 스타일, 타이포그래피 적용
    - 각 페이지의 특성에 맞는 레이아웃 및 콘텐츠 가독성 개선
    - 코드 일관성 확보 (CSS 통합)
- **6차:** 신규 기능 추가 (오늘의 행운 번호)
